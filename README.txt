See the on-wiki documentation (http://meta.wikimedia.org/wiki/Countervandalism_Network/Bots#SWMTBot) for help.

Console.msgs is not compatible with the Console.msgs from CVUBot.
Additionally, please make sure you upgrade Console.msgs if you upgrade SWMTBot.exe.

To install, please edit SWMTBot.ini (follow all instructions; fully-qualified file paths must be specified)
Then run SWMTBot.exe, either by running the executable, or by running "mono SWMTBot.exe", depending on your environment.
